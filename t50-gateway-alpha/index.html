Important Announcement ⚠️

April 2, 2024

Evox Project Status

  • The development team behind Evox has recently encountered challenges with the code testing process.
  • As a result, updates and the introduction of new features for Evox will be temporarily suspended until these issues are resolved.

Evox Datacenter Change

  • Starting today, Evox Datacenter will stop updating to new Evox compatible versions meaning that upcoming features planning will be changed.
  • User data will remain secure, and all Evox operations will continue to function smoothly for all users during this period.

Desktop Application

  • Fortunately, development will persist for the Evox Desktop Application, with enhancements including new features, leveling systems, and further optimization for desktop usage!
  • To access the latest Evox Application, simply log in to your Evox account from a computer and click on the PC icon located in the top right corner. This will enable you to download the updated version seamlessly.

We regret to announce the temporary pause of the Evox project for mobile devices. Rest assured, we are actively working to resolve these issues as swiftly as possible. Thank you for your patience and understanding.

Εκκινηση του project Saint-Paul Shop

Ξεκινωντας το project Saint-Paul Shop, εχουμε ως στοχο να φτιαξουμε ενα σταθερο website αγοραζοντας ισχυρους διακομιστες που θα δεχονται και θα επεξεργαζονται τις ολες τις παραγγελειες αυτοματα.

2η φάση

Χτίσιμο Κοινότητας

Θελουμε με το Saint-Paul Shop να χτισουμε μια κοινοτητα ανθρωπων που μπορουν να λαμβανουν διαφορα προιοντα μεσα στο σχολειο, ευκολα, κρατωντας την ταυτοτητα τους ανωνυμη

3η φάση

Μενοντας κρυφοι

Ειναι σημαντικο για τους πελατες να ακολουθησουν προσεκτικα τους κανονες που δινονται κατα τη παραγγελεια διαφορετικα θα υπαρξουν πιθανοτητες να καταστραφει η ανωνυμοτητα μας και σας

FAQ

Εχετε ερωτησεις?

Ε1. Ειναι η υπηρεσία μας παρανομη?

Οχι. Η διακινιση προιοντων απο μαθητη προς μαθητη εντος σχολειου δε θεωρειται απαραιτητα παρανομη. Μπορουμε απλα να θεωρησουμε πως ειναι σαν ενας γνωστος ενος μαθητη να του δινει ενα προιον σε εκεινον με αντάλλαγμα ενα χρηματικο ποσο. Παρολα αυτα ειναι καλο να κρατησουμε ολες τις συναλλαγες κρυφες.

Ε2. Πως αγοραζω ενα προιον?

Τα βηματα ειναι πολυ απλα! Διαλεγετε ενα προιον, Επιλεγετε τον επιθυμητο τροπο πληρωμης (μετρητα, paypal κτλπ) και Τελος μετα απο ενα μικρο χρονικο διαστημα λαμβανετε ολες τις πληροφοριες που χρειαζεστε (Ημερα παραλαβης, Τελικη Τιμη, Οδηγιες Παραλαβης)

Ε3. Αλλαξα γνωμη και θελω να ακυρωσω την παραγγελεια μου. Πως γινεται?

Η παραγγελεια σας παιρναει απο 3 φασεις. 1. Επεξεργασια 2. Αγορα Προιοντος 3. Παραλαβη απο εσας. Εαν η παραγγελεια σας βρισκεται πριν την φαση 2, γινεται ευκολα να ακυρωθει και να γινει αμεση επιστροφη χρηματων.
Για να ακυρωσετε την παραγγελεια σας παρακαλω επικοινωνστε μαζι μας πατωντας εδω

Ε4. Με βρηκαν καθηγητες με ενα προιον που αγορασα απο εσας. Τι κανω?

Πολυ απλα λετε πως αγορασατε εσεις το προιον απο ενα κοντινο περιπτερο/mini market. Εαν το προιον που αγορασατε εχει απαγορευμενη χρηση απο ανηλικους (18+), πρεπει δυστυχως να λαβετε τις επιπτωσεις καθως και την ποινη που δινει το σχολειο.
